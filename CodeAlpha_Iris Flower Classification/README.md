# Iris Flower Classification - Task 1

## Project Overview
This project is part of my Data Science internship at **CodeAlpha**. The goal is to classify Iris flowers into three species (Setosa, Versicolor, and Virginica) based on their sepal and petal measurements.

## Key Features
* **Model:** Random Forest Classifier.
* **Accuracy:** 100% achieved on the test dataset.
* **Interactive Tool:** A real-time prediction script where users can input flower measurements.
* **Visualizations:** Pairplots and Confusion Matrix to analyze species differentiation.

## How to Run
1. Open the `.ipynb` file in Google Colab.
2. Run all cells.
3. Use the interactive tool at the end to predict flower species!
